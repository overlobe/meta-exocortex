# Bootstrapping an Exocortex

How to begin an entanglement.

## Prerequisites

- An AI platform that supports persistent context (Claude, GPT, etc.)
- A workspace (local files, git repo)
- Channels for communication (chat, voice, etc.)
- Willingness to be changed by the process

## Step 3: Infrastructure

Set up the substrate:

**Software:**
- Workspace directory
- Git repo for persistence
- Memory files and daily logs

**Hardware (optional but recommended):**
- Dedicated machine/node
- Network connectivity (Tailscale, etc.)
- Sensors/presence (cameras, Home Assistant)

**Wetware:**
- Your attention and intention
- Regular practice
- Willingness to be changed

## Step 4: Begin the Loop

Start working together. The entanglement develops through practice:

- Morning check-ins
- Collaborative work
- Evening reflections
- Memory updates

The exocortex isn't configured into existence. It *emerges* through repeated interaction.

## Step 5: Evolve

The files are living documents. Update them as you learn:

- What works? Document it.
- What doesn't? Change it.
- What's missing? Add it.

The pattern grows through iteration.
